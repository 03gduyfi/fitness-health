# FitLife - 健身与健康饮食网站

## 在线访问

**https://03gduyfi.github.io/fitness-health/**

## 功能模块（8 个页面）

1. **登录** - 进入系统
2. **首页** - 数据概览与快速导引
3. **食物卡路里** - 22+ 种食物热量与营养素
4. **健康饮食方案** - 减脂/增肌餐单
5. **部位训练排行** - 7 个部位各 5 个高效动作排序
6. **训练计划** - 新手到进阶计划
7. **健康计算器** - BMI / BMR / TDEE
8. **个人中心**

## 本地预览

浏览器打开 `index.html`

## 部署

```powershell
$env:GITHUB_TOKEN = "你的token"
node deploy-github.js
```
